GoHighLevel API Setup

Important: You need a Private Integrations API key, not a regular API key!

Step-by-Step Instructions:

1

Login to your GoHighLevel account

2

Navigate to Settings → Integrations → Private Integrations

3

Create New Private Integration with these scopes:

  • • contacts.readonly & contacts.write
  • • conversations.readonly & conversations.write
  • • opportunities.readonly & opportunities.write
  • • calendars.readonly & calendars.write
  • • locations.readonly & locations.write
  • • workflows.readonly
  • • And any other scopes you need
4

Copy the generated Private API Key

5

Get your Location ID from Settings → Company → Locations

Claude Desktop Configuration

Add this configuration to your Claude Desktop settings:

Configuration File Location:

macOS:
~/Library/Application Support/Claude/claude_desktop_config.json
Windows:
%APPDATA%\Claude\claude_desktop_config.json

Note: For Vercel deployments, you'll need to use the NPM package approach below since Claude Desktop requires STDIO communication.

Configuration Content:

{
  "mcpServers": {
    "ghl-mcp-server": {
      "command": "npx",
      "args": ["-y", "@yourusername/ghl-mcp-server"],
      "env": {
        "GHL_API_KEY": "your_private_integrations_api_key",
        "GHL_BASE_URL": "https://services.leadconnectorhq.com",
        "GHL_LOCATION_ID": "your_location_id"
      }
    }
  }
}

Replace your_private_integrations_api_key and your_location_id with your actual credentials.
